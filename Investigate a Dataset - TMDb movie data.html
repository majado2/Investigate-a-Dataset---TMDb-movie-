Introduction¶

This data set contains information about 10,000 movies collected from The Movie Database (TMDb), including user ratings and revenue.

Certain columns, like cast and genres, contain multiple values separated by pipe (|) characters. There are some odd characters in the cast column. Dont worry about cleaning them. You can leave them as is. The final two columns ending with _adj show the budget and revenue of the associated movie in terms of 2010 dollars, accounting for inflation over time.

Dataset Description¶

# Column description
0 id primary key for table
1 imdb_id Movie number ID imdb
2 popularity The popularity of the Movie based on views and ratings for more here https://developers.themoviedb.org/3/getting-started/popularity
3 budget The cost of producing the Movie
4 revenue The movie's total revenue
5 original_title Movie title on release date
6 cast The name lead and supporting actors
7 homepage link URL for homepahe for move
8 director director of the movie
9 tagline Brief description of a Movie in one line
10 keywords Keywords for search engines
11 overview Overview of the Movie - a brief description
12 runtime the show length
13 genres Movie type or genre
14 production_companies Movie producing companies
15 release_date Show start date
16 vote_count The number of people who voted
17 vote_average Vote rate
18 release_year The year the Movie was released
19 budget_adj the movie budget, adjusted for inflation
20 revenue_adj the movie revenue, adjusted for inflation
